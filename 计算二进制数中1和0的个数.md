**单纯计算一的个数**
```
int numOfOne(x){
int count = 0;
while(x)
    {count++;
     x = x&(x-1); }return count;
}
```
**单纯计算0的个数**
```
int numOfZero(x){   
while(x+1)
    {count++;
     x = x|(x+1); }return count;
}
```
