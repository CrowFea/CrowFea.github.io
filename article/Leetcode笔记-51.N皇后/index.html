问题描述

n 皇后问题研究的是如何将 n 个皇后放置在 n×n 的棋盘上，并且使皇后彼此之间不能相互攻击。

上图为 8 皇后问题的一种解法。

给定一个整数 n，返回所有不同的 n 皇后问题的解决方案。

每一种解法包含一个明确的 n 皇后问题的棋子放置方案，该方案中 ‘Q’ 和 ‘.’ 分别代表了皇后和空位。

示例:

思路

经典的回溯问题，和数独的题目有点类似。主要就是不能同行同列同斜列。

我们使用一个n*n的表来记录，由上至下由左至右开始选，如果当前位置可以，就将该位置设置为Q，将不能与他重复的位置全置为P。接着递归。当然也可以只标记Q的位置，在判断是否重合的时候用自增自减判断。

要注意的几个点：

  • 回溯前后注意保存现场和恢复现场。我们经常使用引用来传递值，回溯回来后首先要将值还回去，就好像美国队长要把用完的无限宝石还回去，要不然会造成平行宇宙的崩溃。
  • 最好将递归的明显条件写在参数里。i+1这种。
